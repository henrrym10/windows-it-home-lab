# Windows IT Home Lab

## Overview
This project demonstrates a virtual IT environment built to simulate real-world system administration tasks including networking, file sharing, and user permission management.

## Technologies Used
- VMware Fusion
- Windows 11 Pro
- PowerShell
- NTFS Permissions
- Virtual Networking

## Lab Setup
Two virtual machines were created:

DC01 (Server)
CLIENT01 (Client)

## Tasks Completed
- Configured networking between virtual machines
- Tested connectivity using ping
- Created a shared company file server
- Implemented department folders (HR, IT, Finance)
- Created multiple user accounts
- Applied NTFS permissions
- Tested access from client machines
